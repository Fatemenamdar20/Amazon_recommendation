# سیستم پیشنهاددهی محصولات آمازون (Amazon Product Recommendation System)

## توضیح پروژه
این پروژه یک سیستم **Content-Based Recommendation** برای محصولات آمازون است که بر اساس توضیحات محصول (`reviews.text` و `name`) و دسته‌بندی (`categories`) مشابه‌ترین محصولات را به مشتری پیشنهاد می‌دهد.  
هدف این پروژه:
- ارائه پیشنهادات دقیق و مرتبط به مشتری
- استفاده از متن توضیحات و دسته‌بندی‌ها برای استخراج ویژگی‌های محصول
- محاسبه **Cosine Similarity** بین بردارهای متنی برای یافتن محصولات مشابه

این سیستم برای کاربردهایی مثل افزایش فروش، نمایش محصولات مرتبط در وبسایت و بهبود تجربه کاربری طراحی شده است.

## مراحل پردازش و آماده‌سازی
1. **پاکسازی متن (`reviews.text` و `name`)**  
   - حذف HTML، لینک‌ها و نویزهای اضافی  
   - تبدیل متن به حروف کوچک  
   - در صورت نیاز، Lemmatization و Tokenization  

2. **پردازش `categories`**  
   - حذف نویز و استانداردسازی فرمت  
   - استفاده از دسته‌بندی به‌عنوان **فیلتر** در محاسبه شباهت  

3. **بردارسازی محصولات (Vectorization)**  
   - تبدیل متن به بردار با استفاده از TF-IDF یا مدل‌های embedding مانند SBERT  

4. **محاسبه شباهت و پیشنهاد محصول**  
   - محاسبه Cosine Similarity بین بردارها  
   - ارائه Top-K محصولات مشابه برای هر محصول
